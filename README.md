# Java Puzzle Game

Proyecto personal desarrollado durante la asignatura **Programación II** del Grado en Ingeniería Informática.

## Descripción
Juego de puzzle en Java que permite mezclar y resolver piezas de una imagen.  
El objetivo es ordenar todas las piezas correctamente en el menor tiempo posible.  
Este proyecto practica lógica, programación orientada a objetos y manejo de interfaces gráficas con Swing.

## Funcionalidades
- Mezcla automática de las piezas del puzzle
- Botones para **mezclar**, **resolver** y **salir** del juego
- Temporizador que mide el tiempo de resolución
- Interfaz gráfica desarrollada en Java Swing

## Tecnologías
- Java
- Swing
- Programación orientada a objetos (POO)
